# E-commerce Dashboard

1. ( HEADLINE )
= E-Commerce-Sales-Insights-Analysis = This is end-to-end Power BI project transforming messy E-commerce datasets into business-ready insights using Power Query, DAX, and Data Modeling

2. ( PURPOSE )
= This project demonstrates a complete ETL pipeline. 
= I generated two "dirty" datasets—Customer and Orders—and used Power BI to clean, model, and visualize the data to help a business track sales performance and customer demographics

3. ( TECH STACK )
* Excel: Initial data cleaning, blank cell identification, and duplicate removal.
* Power Query (ETL): Standardized names, fixed email syntax, and resolved complex date format issues.
* Data Modeling: Established a One-to-Many relationship between Customers (Primary Key) and Orders (Foreign Key).
* DAX (Data Analysis Expressions): Created measures including DISTINCTCOUNT for customers and CALCULATE with FILTER for specific order statuses.

4. ( Data Source )
Source = Google. 

5. ( BUSINESS INSIDE )
= Order Fulfillment: Tracked the volume of "Completed Orders" vs. total orders.
= Product Performance: Identified "Premium Items Sold" (Unit Price > 100) and analyzed product category distributions.
= Trends: Visualized monthly sales trends to identify peak performance periods.
